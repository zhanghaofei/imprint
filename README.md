# 印记imprint

## 功能展望

- [ ] 导出为pdf、html
- [ ] 自定义markdown样式
- [ ] 语法搜索、展示、插入
- [ ] 支持git，可查看历史记录及对比
- [ ] 多人协作

> An electron-vue project

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


# lint all JS/Vue component files in `src/`
npm run lint

```
